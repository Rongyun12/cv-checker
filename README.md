# CVScore — Testez votre CV face aux robots ATS

**[→ Essayer gratuitement](https://rongyun12.github.io/cv-checker/)**

75 % des CV sont écartés par des logiciels de tri automatique (ATS) avant même d'être lus par un recruteur. CVScore analyse votre CV et vous donne un **score sur 100** avec des conseils concrets pour passer les filtres de Workday, Taleo, SuccessFactors et autres.

## Fonctionnalités

- **Score ATS sur 100** décomposé en 7 critères : coordonnées, sections standard, longueur, verbes d'action, résultats chiffrés, lisibilité machine, mots-clés
- **Revue ligne par ligne** de votre CV avec les points à corriger
- **Reformulations avant/après** tirées directement de vos propres phrases
- **Matching d'offre d'emploi** : collez une offre, voyez les mots-clés manquants
- **Formats acceptés** : PDF, DOCX, TXT — français ou anglais

## Confidentialité totale

L'analyse s'exécute **100 % dans votre navigateur**. Votre CV n'est jamais téléversé, stocké ni transmis à un serveur. Aucun compte, aucun cookie, aucune donnée collectée.

## Technique

Site statique sans backend : HTML/CSS/JavaScript, [pdf.js](https://mozilla.github.io/pdf.js/) pour l'extraction PDF et [mammoth.js](https://github.com/mwilliamson/mammoth.js) pour les DOCX. Hébergé sur GitHub Pages.
